# Project 2: Interactive Financial Dashboard & Slicers

## Overview

This project focuses on turning raw, unformatted financial transaction logs into an executive, interactive **Financial Dashboard** using native Microsoft Excel features without writing manual code.

## Key Features & Business Impact

- **Dynamic Pivot Tables:** Automatically aggregate revenues, costs, and calculate profit margins by category and product.
- **Interactive Slicers:** Implemented visual button filters so stakeholders can instantly filter the entire dataset with a single click.
- **Executive Summary:** Provides quick insights for decision-making and performance tracking.

## Directory Structure

- `data/`: Contains the original unformatted financial CSV file (`financial_raw.csv`).
- `financial_dashboard.xlsx`: The main Excel workbook containing the Pivot Tables, charts, and interactive Slicers.
